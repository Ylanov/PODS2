# Карты зон ответственности — автономная версия

Самодостаточное приложение с **двумя картами**, работает на одном компьютере, **без логинов и учёток**:

- **Карта ОД** — базовая точка, поиск адреса, маршрут от базы, зоны ответственности.
- **Карта зон** — импорт координат из **Excel/Word**, пересчёт **МСК-77 → WGS-84**, таблица координат, подвижка контура, экспорт карты в **JPG/PDF (масштаб 1:2000)**.

Подложка карты, поиск и подсказки — **Яндекс.Карты** (если задан ключ, см. ниже) либо **OpenStreetMap** (без ключа, как запасной вариант). Маршрут — OSRM, рисуется поверх подложки. База — **SQLite** в `%LOCALAPPDATA%\Karty\maps.db` (создаётся сам, переживает перезагрузки). Нужен **интернет** на этом ПК.

## Яндекс.Карты (рекомендуется)
Чтобы карта, поиск адресов и подсказки были **как в основной системе (Яндекс)**, задай ключ Яндекс.Карт одним из способов:

- создай файл **`yandex.key`** рядом с `server.py` и вставь в него ключ одной строкой; **или**
- задай переменную окружения `YANDEX_MAPS_API_KEY` перед запуском.

Ключ — тот же, что в основном проекте (`.env` → `YANDEX_MAPS_API_KEY`). Если ключа нет — приложение автоматически работает на OpenStreetMap (поиск через Nominatim, без подсказок). Какой режим активен — видно в консоли при старте и в подложке карты.

> Маршрут строится через OSRM и рисуется поверх Яндекс-карты — точно как в исходной «Карте ОД» (у Яндекса нет бесплатного API маршрутов под обычный ключ Геокодера).

## Установка как программа «Карты» (рекомендуется)

Получаем привычный **установщик** (мастер с выбором папки) и ярлык — без командной строки.

**Собрать установщик (один раз, на машине разработчика):**
1. Установи [Python 3.10+](https://www.python.org/downloads/) («Add Python to PATH») и бесплатный [Inno Setup 6](https://jrsoftware.org/isdl.php).
2. (Для Яндекса) положи ключ в файл `yandex.key` рядом со скриптами — он попадёт в установщик.
3. Двойной клик по **`build_installer.bat`**.
4. Появится **`Output\Карты-Setup.exe`** — это и есть установочный файл.

**Установка на любой ПК (для пользователя):**
1. Запусти **`Карты-Setup.exe`** → мастер: выбери папку → «Установить».
2. Создаётся ярлык **«Карты»** (рабочий стол + меню Пуск). При желании — галочка «запускать при входе в Windows».
3. Дальше просто **двойной клик по ярлыку «Карты»** → откроется окошко и браузер с картами. Командная строка не нужна, переустанавливать после перезагрузки не нужно.

Данные (зоны, базовая точка, кеш, ключ) хранятся в `%LOCALAPPDATA%\Karty` и **сохраняются между запусками**. Ключ Яндекса можно также ввести прямо в окне программы (кнопка «Сохранить»).

> Готовый `Карты-Setup.exe` содержит ваш ключ Яндекса (если клали `yandex.key`) — не выкладывайте его публично.

## Запуск без установки (для разработки)

- **`run.bat`** — поднимет сервер из исходников (нужен Python). Браузер откроется на http://127.0.0.1:8077.
- Консоль: `python server.py` (Linux/macOS: `bash run.sh`).
- **`build_exe.bat`** — простой одиночный `dist\KartyZon.exe` (с консолью, без установщика).

## Как пользоваться

**Импорт зон (Карта зон):**
1. Выбери «Систему координат»: `WGS-84` (широта/долгота) или `МСК-77, зона 1` (X/Y в метрах).
2. Нажми **Excel** или **Word**, выбери файл.
   - Excel: колонки «Зона», «Широта», «Долгота» (или X/Y для МСК). Кнопка **Шаблон** скачает образец.
   - Word: первая таблица с колонками X/Y (как в актах межевания).
3. Зоны лягут на карту; в таблице слева — исходные координаты и пересчитанные WGS-84.
4. Если есть небольшой остаточный сдвиг — поправь **стрелками** в блоке «Подвижка контура» (или попробуй другой вариант ключей МСК-77).

**Экспорт:** задай масштаб (1:2000), лист (A4/A3), DPI → **JPG** или **PDF**.

## Настройки (необязательно)
Переменные окружения перед запуском:
- `PORT=9000` — другой порт.
- `HOST=0.0.0.0` — доступ с других машин в сети (по умолчанию только localhost).

## Где данные
Папка `%LOCALAPPDATA%\Karty` (у каждого пользователя своя):
- `maps.db` — зоны и базовая точка (SQLite).
- `tiles/` — кеш тайлов карты.
- `yandex.key` — ключ Яндекса, если введён в окне программы.

Удалить эту папку = сбросить всё к чистому состоянию.
